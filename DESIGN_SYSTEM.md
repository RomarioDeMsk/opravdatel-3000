# Дизайн-система "Оправдатель 3000"

## Обзор

Современный, живой и эмоциональный визуальный стиль, создающий положительные эмоции и повышающий вовлеченность пользователей.

## Эмоциональный посыл

- **Радость** — яркие цвета и плавные анимации
- **Легкость** — мягкие тени и скругления
- **Удивление** — неожиданные микроинтеракции
- **Дружелюбие** — теплая палитра и органические формы

## Цветовая палитра

### Основные цвета
- **Primary (Индиго)**: `#6366f1` — основной акцент
- **Secondary (Розовый)**: `#ec4899` — эмоциональный акцент
- **Accent (Изумрудный)**: `#10b981` — успех/действие
- **Warning (Янтарный)**: `#f59e0b` — предупреждение
- **Info (Синий)**: `#3b82f6` — информация

### Градиенты
- **Primary Gradient**: Индиго → Фиолетовый
- **Secondary Gradient**: Розовый → Красный
- **Accent Gradient**: Синий → Голубой
- **Warm Gradient**: Розовый → Желтый
- **Cool Gradient**: Бирюзовый → Фиолетовый
- **Rainbow Gradient**: Многоцветный градиент

### Фоны
- **Primary**: `#0f0f23` — темный основной
- **Secondary**: `#1a1a2e` — темный вторичный
- **Elevated**: `#16213e` — приподнятые элементы
- **Glass**: Полупрозрачный с размытием

## Типографика

### Шрифты
- **Основной**: Inter — для интерфейса
- **Акцентный**: Space Grotesk — для заголовков и специальных элементов

### Размеры
- Заголовки: `clamp(2.5rem, 8vw, 4rem)`
- Подзаголовки: `1.5rem`
- Основной текст: `1rem`
- Мелкий текст: `0.875rem`

## Компоненты

### Кнопки
- **Primary**: Градиент индиго-фиолетовый
- **Secondary**: Градиент синий-голубой
- **Absurd**: Градиент розовый-красный (с эффектом тряски)
- **AI**: Градиент розовый-желтый

**Состояния:**
- Hover: Подъем на 2px, увеличение тени
- Active: Нажатие с масштабированием
- Disabled: Пониженная прозрачность

### Карточки
- Стекломорфизм (backdrop-filter blur)
- Мягкие тени
- Плавные переходы при наведении
- Биоморфные скругления

### Терминал
- Темный фон с градиентной рамкой
- Анимированная сканирующая линия
- Эффект печатной машинки
- Мигающий курсор

## Анимации

### Микроанимации
- **Bounce**: Прыгающие иконки
- **Float**: Плавающие элементы
- **Pulse**: Пульсирующие индикаторы
- **Shake**: Тряска для абсурдных элементов

### Переходы
- **Fast**: 150ms — мгновенные реакции
- **Base**: 300ms — стандартные переходы
- **Slow**: 500ms — плавные изменения
- **Bounce**: 600ms — упругие анимации

### Easing
- `cubic-bezier(0.4, 0, 0.2, 1)` — стандартный
- `cubic-bezier(0.68, -0.55, 0.265, 1.55)` — упругий

## Эффекты

### Неоморфизм
- Мягкие тени с инсет-эффектами
- Объемные элементы

### Стекломорфизм
- Полупрозрачные фоны
- Backdrop-filter blur(20px)
- Тонкие границы

### 3D Эффекты
- Мягкие тени для глубины
- Градиенты для объема
- Transform для перспективы

### Биоморфизм
- Органические формы
- Плавные скругления
- Анимированные фоновые элементы

## Робот Чейми

### Состояния
- **Idle**: Спокойное плавание
- **Thinking**: Покачивание с градиентом акцента
- **Happy**: Увеличение с теплым градиентом
- **Typing**: Подпрыгивание с розовым градиентом
- **Surprised**: Вращение с радужным градиентом

### Анимации
- Автоматическое моргание глаз
- Плавающее движение
- Реакция на hover

## Адаптивность

### Breakpoints
- **Desktop**: > 768px — полная функциональность
- **Tablet**: 481px - 768px — адаптированная сетка
- **Mobile**: ≤ 480px — упрощенный интерфейс

### Адаптивные элементы
- Гибкая типографика (clamp)
- Адаптивная сетка (grid auto-fit)
- Масштабируемые компоненты

## Производительность

### Оптимизации
- CSS-переменные для быстрой смены темы
- Transform вместо position для анимаций
- Will-change для сложных анимаций
- Минимальное использование box-shadow

### Best Practices
- Плавные 60fps анимации
- Оптимизированные градиенты
- Эффективное использование backdrop-filter

## Доступность

- Контрастность текста: WCAG AA
- Фокус-индикаторы для навигации с клавиатуры
- Семантическая разметка
- Альтернативный текст для иконок

## Расширяемость

Дизайн-система легко расширяется:
- Новые цвета через CSS-переменные
- Новые компоненты следуют установленным паттернам
- Анимации переиспользуются через классы
- Градиенты комбинируются для новых эффектов
