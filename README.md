# Оправдатель 3000

Статичный веб-сайт-генератор отговорок для любых жизненных ситуаций.

## Описание

Оправдатель 3000 — это полностью статичный веб-сайт без фреймворков, который помогает генерировать смешные и абсурдные отговорки. Сайт работает полностью на клиенте, используя только HTML, CSS и Vanilla JavaScript.

## Функциональность

- **Случайные отговорки** — быстрая генерация случайной отговорки из базы данных
- **Категории** — выбор отговорок по категориям (Работа, Учеба, Друзья, Семья, Свидание, Универсальные)
- **Абсурдный режим** — генерация полностью абсурдных отговорок путем комбинирования шаблонов
- **Умный помощник** — клиентская логика, имитирующая ИИ для генерации отговорок на основе описания ситуации
- **Коллекция** — сохранение понравившихся отговорок в локальное хранилище
- **PWA** — возможность установки как приложение и работа оффлайн

## Структура проекта

```

```

## Установка и запуск

### Локальный запуск

1. Клонируйте репозиторий или скопируйте файлы
2. Запустите локальный сервер:

```bash
# С помощью Python
python -m http.server 8000

# Или с помощью Node.js (http-server)
npx http-server -p 8000

# Или с помощью PHP
php -S localhost:8000
```

3. Откройте браузер и перейдите по адресу `http://localhost:8000`

### Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы из папки `web/static/` в корень репозитория
3. В настройках репозитория включите GitHub Pages
4. Сайт будет доступен по адресу `https://username.github.io/repository-name`

### Деплой на Netlify

1. Зарегистрируйтесь на [Netlify](https://www.netlify.com/)
2. Перетащите папку `web/static/` в область деплоя
3. Сайт автоматически задеплоится и будет доступен по адресу `https://random-name.netlify.app`

### Деплой на Vercel

1. Установите Vercel CLI: `npm i -g vercel`
2. В папке `web/static/` выполните: `vercel`
3. Следуйте инструкциям

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — стили с использованием CSS Grid, Flexbox, анимаций
- **Vanilla JavaScript (ES6+)** — модульная архитектура без фреймворков
- **Service Worker** — для PWA и оффлайн-работы
- **LocalStorage** — для хранения коллекции и истории

## Особенности

- Полностью статичный сайт — не требует сервера
- Работает оффлайн после первой загрузки
- Адаптивный дизайн — работает на всех устройствах
- Яркий пиксельный дизайн в стиле ретро-футуризма
- Анимированный робот-талисман Чейми
- Эффект печатной машинки при выводе отговорок

## Расширение функциональности

### Добавление новых отговорок

Отредактируйте файл `data/excuses.json` и добавьте новые объекты в массив `excuses`:

```json
{
  "id": 31,
  "category": "work",
  "text": "Ваша новая отговорка здесь",
  "isAbsurd": false
}
```

### Добавление шаблонов для абсурда

Отредактируйте файл `data/templates.json` и добавьте новые элементы в массивы `part1`, `part2`, `part3`.

### Изменение дизайна

Все стили находятся в файле `style.css`. Используйте CSS-переменные в `:root` для быстрой смены цветовой схемы.
